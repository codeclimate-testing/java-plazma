Создание проекта



Создание нового проекта

Для создания нового проекта Вам необходимо выбрать пункт системного меню:
File -> New -> Project...

В появивишимся диалоговом окне выбрать Java Project и нажать Next.

Далее необходимо указать имя проекта и завершить процес кнопкой Finish


Создание проекта с помощью импорта

Если исходный код загружен в виде архива, то вначале архив необходимо распаковать, а потом импортировать каждый проект в Eclipse IDE.

Для выполнения импорта необходимо выбрать пункт меню File -> Import...

Далее необходимо выбрать вариант импорта Existing Projects into Workspace и перейти на следующую страницу нажав Next

Теперь необходимо указать директорию из которой будет импортирован проект и завершить процес нажав на Finish. При выборе варианта Copy projects into workspace проект будет скопирован в Workspace


Создание проекта из SVN

Для начала необходимо создать подключение к SVN. Выберите в системном меню пункт Windows -> Open Perspective и откройте перспективу SVN Repository Exploring

В данной перспективе необходимо выбрать представление SVN Repositories

Теперь можно создать новый репозиторий нажав на кнопку Add SVN Reporsitory в панеле инструментов данного представления.

Укажите параметры доступа к SVN и завершите процес кнопкой Finish

Репозиторий готов к работе. Список всех проектов можно просмотреть в разделе trunk.

Далее необходимо выполнить команду Check Out для каждого проекта.

Команда Check Out загружает проект в Workspace.

Для продолжения работы над проектами перейдите в переспективу Java

Для навигации по проектам можно использовать представление Navigator или Package Explorer
