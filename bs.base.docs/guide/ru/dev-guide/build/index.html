Сборка проектов



Plazma System состоит из двух частей:

  • Plazma Framework
  • Plazma Business Solution

Каждая часть представлена своими проектами.

Для каждого проекта происходит отдельный процесс сборки.

Для сборки проекта используется Apache Ant. Инсталляцию и необходимую документацию Вы можете загрузить из сайта http://ant.apache.org

В корневой папке каждого проекта находятся два файла: 


  build.properties - файл свойств
  build.xml - файл выполняемых задач (task)

Детальнее получить информацию о task'ах фреймворка Ant можно из документации на сайте разработчика.

Для запуска задач необходимо в корневой папке проекта выполнить следующую команду: 


  ant <task-name>
где <task-name> название задачи, например: 

  ant build

Название Описание

clean Очистка результирующих папок (classes и т.д.)

convert Конвертация файлов свойств (.properties) из текущей кодовой страницы (например cp1251) в юникод
create-jar-all Создание всех jar файлов
create-jar-common Создание jar файла для общей части
create-jar-launcher Создание launcher jar файла
