Create a project



Create new project

To create new project you have to select menu:
File -> New -> Project...

Then select Java Project and press Next.

Next you have to input name of the project and press Finish


Create a project by import

If you download source code as zip archive, you have to unzip archive and import each project to the Eclipse IDE.

To import you have to select menu File -> Import...

Then you have to select type of import Existing Projects into Workspace and press Next

Now you have to input an import folder and press Finish. If you check Copy projects into workspace project will be copy to Workspace


Create new project from SVN

First you have to create the connection to SVN. Select menu Windows -> Open Perspective and open perspective SVN Repository Explorirng

Tnen you have to select view SVN Repositories

Now you can create new repository: Add SVN Reporsitory

Input SVN repository setting and press Finish

The repository is ready. You can see the project list in folder trunk.

Then you have to execute command Check Out for each project.

Command Check Out loads a project to workspace.

To work with projects you can select perspective Java

To projects navigation you can use views Navigator or Package Explorer
