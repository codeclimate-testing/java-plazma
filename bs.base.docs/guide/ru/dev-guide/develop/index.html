Модули в SVN (/trunk):

bs.base - BSolution common, server (бины/персистенс объекты, сервисы)
bs.base.client.swt - BSolution SWT клиентская часть
bs.base.cleint.swing - BSolution Swing клиентская часть (низкий приоритет)

bs.base.configuration - BSolution конфигурация
bs.base.db - BSolution база данных, скрипты, тестовые данные
bs.base.reportstorage - BSolution Хранилище отчётов

framework - Framework общий
lib - Общие .jar файлы
framework.client - Framework для создания Client
framework.client.swt - Framework для создания SWT Client
framework.client.swing - Framework для создания Swing Client

Инструменты для работы

  • Java SDK (J2SE 1.5 или выше)
  • IDE (Интегрированая среда разработки)
Теоретически, возможна разработка без использования IDE с приминением любого текстового редактора, но IDE значительно ускорит и упростит весь процес разработки.
Если Вы всё таки отказались от использования IDE, то хорошим помощником в разработке будет Apache Ant.
Ant зачастую входит в комплект поставки современных средств разработки. В качестве IDE используются следующие варианты:
  • Eclipse
  • NetBeans
  • IDEA
Это делеко не полный список, можна использовать любую другую IDE для разработки программного обеспечения на Java.
