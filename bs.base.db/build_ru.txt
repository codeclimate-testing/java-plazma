
ANT задачи для сборки проекта
==================================================================================================


clean-script-result			     Очистка папки с результирующими SQL скриптами 



script-create-db                 Генерация SQL скрипта для создания базы данных (tables, sequences, constraints, views) 

script-create-views              Генерация SQL скрипта для создания вьюверов (views) в базе данных 

script-init-db                   Генерация SQL скрипта для заполнения базы данных инициализирующей информацией 

script-populate-db               Генерация SQL скрипта для заполнения базы данных стартовой информацией (не обязательно)



script-result-all-dbs            Генерация результирующих SQL скриптов для создания баз данных (рабочая база и демонстрационная база) 

script-result-db 		         Генерация результирующего SQL скрипта для создания базы данных

script-result-db-demo            Генерация результирующего SQL скрипта для создания демонстрационной базы данных
