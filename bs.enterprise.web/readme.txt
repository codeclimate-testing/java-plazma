
Installation
=================================================================================================================

Set Plazma Platform properties.

For example (Tomcat):

You have to change file 'catalina.bat/sh':

set JAVA_OPTS=%JAVA_OPTS% %LOGGING_MANAGER% -Dplazma.platform.dir=C:\Dir\Plazma\bs.enterprise.configuration

'plazma.platform.dir' is real path to Plazma Configuration directory.
