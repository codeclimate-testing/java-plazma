Термины



Plazma Framework - это платформа на основе которой строятся различные прикладные решения.

Plazma BSolution - это набор прикладных решений (в различных вариантах, конфигурациях и т.д.)

Plazma System - это любой вариант объединения Plazma Framework и Plazma BSolution

Plazma ERP+CRM - это ERP+CRM на базе Plazma System (часный случай Plazma System).
На базе Plazma System можно построить не только ERP или CRM, но и любое другое бизнес решение

Plazma Workspace - это рабочее пространство, директория где размещаются все проекты Plazma System
