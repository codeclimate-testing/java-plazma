Создание базы данных

Создание базы данных: 

 <PostgreSQL>/bin/psql -U postgres

 postgres=# create database plazma;
 postgres=# \q

Создание структуры базы и заполненние инициализирующими данными: 

 <PostgreSQL>/bin/psql -d plazma -U postgres -f с:\Plazma\db\script\createdb\postgresql\ru\create_plazma_db.sql




JDBC драйвер

Необходимо проверить наличие драйвера: 


 <Plazma>/lib/jdbc/postgresql/postgresql-8.0.309.jdbc3.jar

Необходимо проверить правильность установки CLASSPATH для данного драйвера в файле: 


 <Plazma>/plazma.classpath
 ------------------------------------------------------
 # PostgreSQL 
 lib/jdbc/postgresql/postgresql-8.0.309.jdbc3.jar


Подключение к базе данных

Для настройки подключения необходимо изменить соответствующие свойства в файле <Plazma>/conf/jdbc.properties.

Например: 



 # Установка свойств подключения к базе данных
 jdbc.driverClassName=org.postgresql.Driver
 jdbc.url=jdbc:postgresql://localhost:5432/plazma
 jdbc.username=postgres
 jdbc.password=tiger
